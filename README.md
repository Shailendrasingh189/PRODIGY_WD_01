# PRODIGY_WD_01
Car Rental Landing Page

Overview
This project is a landing page for a car rental service, designed to provide an intuitive and visually appealing user interface. The page allows potential customers to explore the services offered, view available cars, and understand the benefits of renting from our service.

Features
Responsive Design: The landing page is fully responsive and works seamlessly on desktops, tablets, and mobile devices.
Modern UI/UX: Designed with a modern and clean user interface to ensure a great user experience.
Interactive Elements: Includes interactive elements such as carousels, hover effects, and animations to engage users.
Contact Form: A functional contact form for users to get in touch with the car rental service.
Service Information: Detailed sections about the car rental process, available cars, pricing, and customer testimonials.
Technologies Used
HTML5: For structuring the content of the web page.
CSS3: For styling the web page, including layout, colors, and animations.
JavaScript: For adding interactivity to the web page, such as form validation and dynamic content.

**Installation**
To run this project locally, follow these steps:

**Clone the repository:**

git clone https://github.com/your-username/PRODIGY_WD_01.git
Navigate to the project directory:

*cd PRODIGY_WD_01 *
Open index.html in your web browser:

*open index.html*

Alternatively, you can use a live server extension in your code editor for a better development experience.

**Usage**

Homepage: The main section with a hero image and a brief introduction to the car rental service.
Services Section: Detailed information about the services offered.
Available Cars: Showcase of the cars available for rent with images and descriptions.
Testimonials: Customer reviews and feedback.
Contact Form: Users can fill out the form to get in touch with the car rental service.
